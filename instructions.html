Instructions for using this app

  • First, go to toolbar and click on "Add on", click the name of the current document , then click "Run"

  • A sidebar will appear on the right of the google document.

  • Click "Help" to view instructions.

  • click corresponding buttons to enter personal information and prepare for generating resumes

  • to remove everything existing in the current doc, use clear doc button

  • Click edit section to display what is already in experience, education, honor, header, skills and free text. Submit changes to the entry you are viewing and click the display button to see changes you made

  • Delete button to delete section info you had entered




Functions, GUI to be implemented

  • showExperienceDialog(), showSkillSetDialog(), showHeaderInputDialog(), showEducationDialog(), showHonorDialog(), showSectionDialog will be called in "script" tag of the corresponding html file to have pop up windows
  • insertTemaplate1(), insertTemaplate2(), insertTemplate3() will insert the template with user info on the google doc
  • setters and getters are the storing and retriving information to and from the global variable in google app script
  • Each pop up window will handle one type of user information
  • graphic requirement is met by functionality in survey.html(self evaluation) and portfolio
